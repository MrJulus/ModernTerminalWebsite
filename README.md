# Command Description
## Available Commands
### `cd <path>`
- Changes the current page by navigating to the specified URL.
- **Example**: `cd about` redirects to `/about`.

### `post new message <message>`
- Adds a new message to the Firebase database.
- **Example**: `post new message Hello` adds "Hello" to the list of messages.

### `get messages`
- Retrieves and displays all stored messages from Firebase.
- **Example**: `get messages` shows the list of stored messages.

### `clear`
- Clears the current console display.
- **Example**: `clear` removes all displayed content.
